# Library Examples

Use NixClaw as a Python library by importing it into your own scripts.

```python
pip install nixclaw
```

## Examples

| File | Topic |
|------|-------|
| `01_basic_usage.py` | Run a one-shot task |
| `02_streaming_output.py` | Real-time agent output |
| `03_direct_agent.py` | Create agents directly from factory |
| `04_team_mode.py` | Multi-agent collaboration |
| `05_custom_tools.py` | Register your own functions as tools |
| `06_task_manager.py` | Task DAGs with dependencies |
| `07_event_bus.py` | Async pub/sub event system |
| `08_api_client.py` | HTTP client for the REST API |
| `09_telegram_notifications.py` | Send Telegram notifications |
| `10_context_manager.py` | Sliding window context with token budget |
| `11_retry_and_error_handling.py` | Exponential backoff retry |
| `12_security_utils.py` | Secret masking, path sanitization |
| `13_health_check.py` | Monitor system components |
| `14_shell_executor.py` | Safe command execution |
| `15_async_task_queue.py` | Background task queue |
| `16_database_persistence.py` | SQLite persistence |
